# 医生端界面重新设计 - Zocdoc 风格

## 设计变更总结

### 🎨 设计理念
参考 Zocdoc 的设计语言,采用简洁、现代、专业的视觉风格。

---

## 主要变更

### 1. 布局结构 ✅
**之前:** 左侧固定侧边栏 + 主内容区
**现在:** 顶部导航栏 + 主内容区

#### 顶部导航栏包含:
- **Logo 区域** - 左侧,可点击返回首页
- **中间导航菜单** - 工作台、预约管理、排班日历、请假管理、我的评价
- **右侧操作区** - 日期显示、通知图标、用户下拉菜单

### 2. 配色方案 🎨
- **主色:** `#FFD300` (黄色) - 用于按钮、强调元素
- **背景色:** `#FAFAFA` (浅灰) - 页面背景
- **卡片背景:** `#FFFFFF` (白色)
- **边框:** `#E8E8E8` (浅灰色边框)
- **文字:** `#2A2A2A` (深灰色主文本)

### 3. 组件风格

#### 统计卡片
```
┌─────────────────────────────┐
│ 今日预约            📅     │
│                              │
│        15                    │
│                              │
│ 待就诊 8 人                 │
└─────────────────────────────┘
```
- 更扁平化的设计
- 图标在右上角
- 数值突出显示
- 底部显示详细信息

#### 日历组件 (Schedule.vue)
**Zocdoc 风格特点:**
```
┌───┬───┬───┬───┬───┬───┬───┐
│ 周日│ 周一│ 周二│ 周三│ 周四│ 周五│ 周六│
├───┼───┼───┼───┼───┼───┼───┤
│ 1 │ 2 │ 3 │ 4 │ 5 │ 6 │ 7 │
│   │ 5 │ 6 │12 │ 8 │   │   │
│   │上午│上午│上午│上午│   │   │
│   │ 3 │ 4 │10 │ 6 │   │   │
│   │下午│下午│下午│下午│   │   │
└───┴───┴───┴───┴───┴───┴───┘
```

**改进点:**
- ✅ 更紧凑的单元格尺寸 (100px height vs 90px)
- ✅ 显示每天的号源数量,而不是简单的"上午/下午"标签
- ✅ 使用浮动面板显示详情,而不是底部固定区域
- ✅ 干净的边框和间距

#### 预约卡片
```
┌─────────────────────────────────────┐
│ 🕐 上午    张三 [待就诊]            │
│            主诉:头痛三天           │
│            [签到] [完成] [爽约]   │
└─────────────────────────────────────┘
```
- 水平布局,信息一目了然
- 状态标签醒目
- 操作按钮小而精致

### 4. 交互优化

#### Hover 效果
- 卡片 hover 时轻微上移 (`translateY(-2px)`)
- 添加阴影增强层次感
- 按钮 hover 时颜色变化

#### 过渡动画
- 所有交互元素添加 `transition: all 0.2s`
- 流畅自然的动画效果
