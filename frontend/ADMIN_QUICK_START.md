# 管理员端快速启动指南

## 🚀 启动步骤

### 1. 确认后端已启动

确保后端服务运行在 `http://localhost:8080`

### 2. 启动前端

```bash
cd frontend
npm run dev
```

前端会运行在 `http://localhost:3000`

### 3. 登录管理员账户

使用管理员账号登录：
- 手机号：`13800000001`
- 密码：`password`（根据实际密码）

### 4. 访问管理后台

登录成功后，会自动跳转到 `/admin/home`

## 🔍 测试步骤

### 测试医生审批

1. 访问 `/admin/doctors`
2. 如果有待审批医生，会显示在列表中
3. 点击 "查看详情" 查看医生资料
4. 点击 "通过审批" 批准注册
5. 成功后会显示成功提示

### 测试科室管理

1. 访问 `/admin/departments`
2. 点击 "添加科室" 按钮
3. 填写科室信息：
   - 科室名称：神经内科
   - 科室分类：内科系
   - 科室位置：门诊大楼五层
   - 联系电话：010-12345678
4. 点击 "确定" 保存
5. 成功后会在列表中显示新科室

### 测试排班管理

1. 访问 `/admin/schedules`
2. 点击 "批量创建排班" 按钮
3. 选择医生
4. 设置日期范围（明天到下周）
5. 选择工作日（周一至周五）
6. 勾选上午和下午，设置号源数量（各20个）
7. 点击 "创建排班"
8. 成功后会显示成功提示

### 测试请假审批

1. 访问 `/admin/leaves`
2. 如果有待审批请假，会显示在列表中
3. 查看请假信息（医生、日期、原因）
4. 点击 "批准" 或 "拒绝"
5. 拒绝时需要填写拒绝原因
6. 成功后会显示成功提示

### 测试用户管理

1. 访问 `/admin/users`
2. 查看所有用户列表
3. 使用筛选功能（角色、状态）
4. 点击 "详情" 查看用户信息
5. 对正常用户点击 "锁定"
6. 确认后用户状态会变为 "已锁定"
7. 点击 "解锁" 恢复正常状态

## ❗ 常见问题

### 1. 登录后没有跳转到管理后台

**原因**：用户角色不是 ADMIN

**解决**：确保使用的是管理员账户（role: 'ADMIN'）

### 2. API请求失败 (404)

**原因**：后端服务未启动或接口路径不正确

**解决**：
1. 确认后端服务已启动
2. 检查浏览器控制台的请求URL
3. 确认后端有对应的接口

### 3. 页面显示空白

**原因**：可能是路由配置或组件加载问题

**解决**：
1. 打开浏览器开发者工具查看错误信息
2. 确认路由配置正确
3. 确认组件文件存在

### 4. 统计数据显示为0

**原因**：数据库中没有数据或API返回失败

**解决**：
1. 检查数据库中是否有数据
2. 查看浏览器控制台是否有API错误
3. 确认后端API正常返回数据
