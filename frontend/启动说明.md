# 🚀 快速启动指南

## 第一步：安装依赖

打开 PowerShell，进入 frontend 目录：

\`\`\`powershell
cd frontend
npm install
\`\`\`

如果 npm 安装很慢，可以使用淘宝镜像：

\`\`\`powershell
npm config set registry https://registry.npmmirror.com
npm install
\`\`\`

## 第二步：启动开发服务器

\`\`\`powershell
npm run dev
\`\`\`

启动成功后，浏览器会自动打开 http://localhost:3000

## 第三步：体验功能

### 登录页面
- 访问 http://localhost:3000
- 选择角色（患者/医生/管理员）
- 输入用户名和密码登录

### 注册页面
- 点击"立即注册"
- 选择身份（患者/医生/管理员）
- 填写信息完成注册

### 患者功能
- 首页：查看快捷入口和待就诊预约

## 常见问题

### 1. 端口被占用

如果 3000 端口被占用，修改 `vite.config.js` 中的端口：

\`\`\`js
server: {
  port: 3001, // 改成其他端口
  open: true
}
\`\`\`

### 2. 后端 API 未启动

目前前端已完成，后端开发中。API 请求会失败是正常的。

### 3. 依赖安装失败

尝试清除缓存重新安装：

\`\`\`powershell
rm -r node_modules
rm package-lock.json
npm install
\`\`\`
